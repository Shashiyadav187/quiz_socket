READ.ME

Este proyecto se ha generado con Angular Full-Stack Generator version 3.0.0-rc4 siguiendo el trabajo propuesto en el MOOC Desarrollo de servicios en la nube con HTML5, Javascript y node.js

Hosting

  • app heroku
  • bbdd mongolab
  • src github

Tecnologías

  • Git - control de versiones
  • Mean Stack - arquitectura de la aplicación
    • MongoDB - bbdd noSQL
    • express - framework para nodejs
    • angular - framework MVC
    • Node.js and NPM >= v0.12.0 - backend de la aplicación
      • socket - Para la comunicación web en tiempo real entre cliente y servidor.
  • Herramientas JS
    • bower - $ npm install --global bower
    • Grunt - $ npm install --global grunt-cli
    • yeoman - Generador de la aplicación
    • mocha - Test unitarios
  • Herramientas CSS
    • bootstrap - framework desarrollo web
    • [Sass] - lenguaje de hoja de estilos
      • Ruby - para instalar sass: $ gem install sass

Desarrollar

  • $ yo angular-fullstack [nombre_app] para generar la aplicación
  • $ npm install para instalar las dependencias
  • $ bower install para instalr las dependencias del front-end
  • $ grunt serve para levantar el servidor de desarrollo

Build

  • $ grunt build genera la version para hacer el deploy, en la carpeta "dist"

Test

  • $ npm test
